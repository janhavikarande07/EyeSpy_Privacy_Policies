Privacy Policy

Effective Date: 06/04/2026

At EyeSpy, your privacy is important to us. This Privacy Policy explains what data we collect, how it is used, and how we protect your information. By using EyeSpy, you agree to the practices described below.

1. Overview

EyeSpy is an educational eye health awareness application designed to help users learn about eye health through visual tools and interactive tests.

EyeSpy does not provide medical advice, diagnosis, or treatment. All features are intended for awareness and educational purposes only.

2. Information We Collect

EyeSpy is designed to minimise data collection.

  • ❌ No account required
  • ❌ No login or personal details
  • ❌ No name, email, or phone number collected
  • ❌ No data linked to any personal identifier

3. Optional User-Provided Information

When generating a report, EyeSpy may allow users to optionally enter details such as name, age, and basic eye or medical history for contextual understanding.

  • This information is entered voluntarily by the user
  • It is used only within the current session to personalise the generated report
  • It is not stored, saved, or transmitted to any server
  • It is not linked to any persistent user profile or identifier
This information is used solely for generating a temporary report and is discarded after use.

4. Images & Camera Usage

When you use the eye scan feature:

  • Images may be temporarily transmitted to secure servers hosted on Google Cloud for real-time processing
  • Images are used only to generate awareness insights within the app
  • Images are not stored, saved, or retained after processing
  • Images are not used for advertising, analytics, or tracking
  • Images are not linked to any personal identifier within EyeSpy
Camera permission is requested only when the user initiates an eye scan. EyeSpy does not access the camera at any other time.

5. Vision & Colour Tests

  • Test responses are processed locally on your device
  • Results are not stored or transmitted to any server
  • These features are intended for educational awareness only

6. How We Use Data

EyeSpy uses temporary data only to:

  • Process images for AI-assisted visual awareness insights
  • Display educational results within the current session
  • Enable app functionality
All data processing is session-based and automatically cleared after use.

7. Use of Device Features

  • Camera → to capture eye images for the awareness scan
  • Photos / Media → to allow users to select images from their device
  • Location → used to display nearby eye care professionals via Google Maps. Location data is handled by Google Maps and is not stored by EyeSpy.

All permissions are requested only when needed and are used solely for the intended feature.

8. Third-Party Websites & Services

EyeSpy may provide access to third-party websites and services (such as eyewear platforms like Lenskart or financial aid resources) through an in-app browser (WebView).

  • These services are external to EyeSpy
  • EyeSpy is not affiliated with, endorsed by, or responsible for third-party content, services, or privacy practices
  • Users are informed when they are viewing external content
  • Google Maps is used solely to display nearby eye care professionals and is subject to Google's Privacy Policy

If any government-related resources are referenced, they are sourced from publicly available information. EyeSpy is not affiliated with any government entity.

9. Data Storage & Security

  • EyeSpy does not store personal or sensitive user data
  • Image processing occurs via secure Google Cloud infrastructure over encrypted HTTPS connections
  • No image data is retained after processing is complete
  • Vision test results are stored only locally on the user's device within the current session

10. Data Sharing

  • EyeSpy does not sell, rent, or share user data
  • No personal data is shared with third parties
  • Eye images and awareness results are never shared externally

11. Data Retention and Deletion

EyeSpy does not store personal data, images, or reports on its servers. Any data such as images or user inputs is processed temporarily in memory only for real-time awareness features and is not retained after processing is complete.

Since no personal data is stored, there is no user data retained by the app that requires deletion. All data is automatically discarded immediately after use.

If users have any questions or requests regarding data handling, they can contact us at: siddhartha.manaktala@gmail.com

12. Medical Data Notice

EyeSpy provides educational awareness insights only and is not a medical device. It does not provide clinical services or medical advice. Users should consult a qualified ophthalmologist or healthcare professional for proper evaluation and care.

13. Children's Privacy

EyeSpy is intended for users aged 13 and above. We do not knowingly collect data from children under 13. EyeSpy may be used in supervised environments such as schools or NGOs, where no personal data is collected from minors.

14. Changes to This Policy

This Privacy Policy may be updated periodically. Any changes will be reflected on this page with an updated effective date. Continued use of EyeSpy after changes constitutes acceptance of the updated policy.
